# Abela Pre & Primary School Frontend
Premium multi-page frontend built with React + Vite + Tailwind.
## Run
npm install
npm run dev
## Build
npm run build
## Deploy to Vercel
- Import repo in Vercel
- Build command: `npm run build`
- Output dir: `dist`
- SPA fallback handled in `vercel.json`
## Editable school details
`src/data/schoolConfig.js`
## Replace images
`src/data/images.js` (Pexels URLs)
## Future backend upgrades
Admissions API, parent portal auth, CMS content management.
